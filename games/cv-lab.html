CV Lab

An interactive computer-vision bench — the classic operations, run by hand with the math on screen. Drop an image, pick a technique, turn the knobs, and watch every intermediate step. Convolution, gradients & edges, Gaussian/Laplacian pyramids, the 2D Fourier transform, JPEG-style DCT, HOG, Harris corners, homographies, and a backprop computation graph. Chain them like layers — hit Edit this → on any result to make it the new working image and run the next operation on top. Everything runs locally in your browser.

Get your games file (Mac)

GamePigeon doesn't expose links in Messages, but they're in your Messages database. In Terminal — which needs Full Disk Access (System Settings → Privacy & Security → Full Disk Access → enable Terminal) — run this. It writes ~/gamepigeon_games.txt; drop that file above.

sqlite3 "$HOME/Library/Messages/chat.db" \
  "SELECT hex(payload_data) FROM message WHERE balloon_bundle_id LIKE '%gamepigeon%' AND payload_data IS NOT NULL ORDER BY date DESC LIMIT 60;" \
| python3 -c 'import sys,plistlib,binascii
for h in sys.stdin:
    try:
        pl=plistlib.loads(binascii.unhexlify(h.strip()))
        u=next((o for o in pl.get("$objects",[]) if isinstance(o,str) and o.startswith("data:?ver=")),None)
        if u: print(u)
    except Exception: pass' \
> ~/gamepigeon_games.txt && open -R ~/gamepigeon_games.txt

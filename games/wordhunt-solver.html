How do I get the link? (Mac)

GamePigeon doesn't show the link in Messages, but it's stored in your Messages database. In Terminal (needs Full Disk Access), run this — the top line is your most recent game. Paste it in the box above.

sqlite3 "$HOME/Library/Messages/chat.db" \
  "SELECT hex(payload_data) FROM message WHERE balloon_bundle_id LIKE '%gamepigeon%' AND payload_data IS NOT NULL ORDER BY date DESC LIMIT 6;" \
| python3 -c 'import sys,plistlib,binascii
for h in sys.stdin:
    try:
        pl=plistlib.loads(binascii.unhexlify(h.strip()))
        u=next((o for o in pl.get("$objects",[]) if isinstance(o,str) and o.startswith("data:?ver=")),None)
        if u: print(u)
    except Exception: pass'
